# Requirements for the scPortrait feature extraction method (method: "scportrait").
#
# scPortrait pins `cellpose>=3.1.1,<4`, which is INCOMPATIBLE with the
# cellpose-sam (cellpose 4.x) stack used elsewhere in this project. It must
# therefore be installed into its own isolated environment, NOT the primary one.
#
# Requires Python 3.11 (scPortrait 1.8.0 does not support 3.10).
#
# Setup (example with uv; plain venv + pip works too):
#   uv venv env-scportrait --python 3.11
#   uv pip install --python env-scportrait/bin/python -r src/feature_extraction/requirements-scportrait.txt
#   uv pip install -e . --python env-scportrait/bin/python   # editable project install for `import src...`
#
# GPU note: the torch wheels below are CUDA 12.8 builds (from the PyTorch index)
# to match the SLURM compute nodes. For CPU-only, drop the --extra-index-url line
# and the "+cu128" local version tags.
